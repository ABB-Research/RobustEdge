# RobustEdge

This repository contains artifacts and code for the paper "Perturbation-Driven Robustness Benchmarking of
ML-Based Detection of Database Write Attacks at
the Industrial Edge", Manca et al., EFTA 2026.

In particular, it contains the following two components:
- [RobustEdgeTestbed](RobustEdgeTestbed): a testbed for generating labeled industrial control system (ICS) telemetry under cyberattack conditions.
- [RobustEdgeBench](RobustEdgeBench): analysis pipeline for benchmarking the robustness of ML-based container attack detection models against perturbations in the ICS telemetry.
